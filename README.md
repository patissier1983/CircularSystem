# LibreFoodSystem

**都道府県規模・循環型社会システムの設計モデル集**  
*A collection of prefectural-scale circular society system design models.*

---

## このリポジトリについて / About

LibreFoodSystemは、食料生産・エネルギー・炭素固定・資源循環を一体設計した都道府県規模の循環型社会システムモデルを収録するリポジトリです。

各地域の農業廃棄物・林業廃材・畜産糞尿・バイオマスエネルギーを組み合わせ、化石資源への依存を段階的に置き換えながら、県域での食料・エネルギー自給と炭素固定を同時に実現する設計思想を共有します。国の政策（みどりの食料システム戦略・地域循環共生圏・国土強靱化・食料安全保障）が個別に掲げる方向性を、各地域の実情に合わせて一つの物理システムへ統合することを目的とします。

This repository collects prefectural-scale circular society system design models that co-design food production, energy, carbon fixation, and material cycling. Each model replaces fossil resource dependency with regional biomass and agricultural residues, simultaneously achieving food and energy self-sufficiency with measurable carbon removal. The design intent is to integrate the directions set separately by national policies into a single physical system adapted to local conditions.

## 設計の共通思想 / Common Design Principles

1. **廃棄物を起点にする** — 誰も欲しがらないものを資源の入口にする
2. **熱カスケードで無駄をなくす** — 廃熱を上位から下位工程へ段階的に使い切る
3. **炭素を製品に閉じ込める** — バイオ炭・バイオプラスチック・CaCO₃として大気に戻さない
4. **収益を生産現場に還す** — Jクレジット・カーボン支援金の帰属を農家・林業者・畜産家に明確化する
5. **単一技術に依存しない** — 複数の技術パスを並走させてリスクを分散する
