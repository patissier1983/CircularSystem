# Biwako System 2.0 / 琵琶湖システム2.0

**都道府県規模・循環型社会モデルの設計図（滋賀県モデル実装例）**  
*A design model for a prefectural-scale circular society system — implemented as the Shiga Prefecture case.*

---

## このリポジトリについて / About

琵琶湖システム2.0は、都道府県規模での循環型社会システムの設計モデルです。稲藁・木質廃材を原料に、バイオマス発電・セルロースエタノール製造・バイオ炭農地施用・CaCO₃循環を一体設計し、県域での食料・エネルギー自給と炭素固定を目指します。本提案は滋賀県を実装例（モデルケース）としており、各地域の資源・産業・制度に合わせて他の都道府県にも適用できる設計思想を持ちます。

Biwako System 2.0 is a design model for a prefectural-scale circular society system. It co-designs biomass power generation, cellulosic ethanol production, biochar soil application, and CaCO₃ cycling from rice straw and woody residues — targeting regional food and energy self-sufficiency with measurable carbon removal. Shiga Prefecture serves as the model implementation case; the underlying design principles are applicable to other prefectures with adaptation to local resources, industries, and institutions.
